# 🌍 AI Village Help Portal

## 📌 Project Overview

AI Village Help Portal is a smart public service platform developed to help village and rural citizens access important government and public services easily.

This application provides guidance for:

* Government schemes
* Aadhaar and PAN services
* Farmer crop support
* Health assistance
* Public complaints
* Education and scholarships
* Jobs and skill training
* Emergency services

The platform is designed using Streamlit and Python.

---

## 🚀 Features

### 🪪 Government Services

* Aadhaar guidance
* PAN card guidance
* Ration card information
* Income, caste, and residence certificates
* Pension schemes

### 🌾 Farmer Support

* Crop guidance
* Fertilizer suggestions
* Disease prevention tips
* Seasonal farming advice

### 🏥 Health Guidance

* Fever support
* Cold and cough remedies
* Headache guidance
* Stomach pain advice
* First aid information

### 🚨 Complaint System

Citizens can report:

* Garbage issues
* Water problems
* Road damage
* Streetlight complaints

### 🎓 Education Support

* Scholarships
* Student schemes
* Skill development programs

### 💼 Job Support

* Rural job information
* Skill training guidance

### ☎ Emergency Services

* Ambulance
* Police
* Fire services
* Women helpline

## ▶ How to Run the Project

### Clone Repository

```bash
git clone https://github.com/meghanareddy0915-ops/ai-village-help-portal.git
```

### Open Project

```bash
cd ai-village-help-portal
```

### Install Requirements

```bash
uv sync
```

### Run Application

```bash
uv run streamlit run app.py
```
