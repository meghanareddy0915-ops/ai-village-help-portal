# 🌍 AI Village Help Portal

## 📌 Project Overview

AI Village Help Portal is a smart public service platform developed to help village and rural citizens access important government and public services easily.

This application provides guidance for:

* Government schemes
* Aadhaar and PAN services
* Farmer crop support
* Health assistance
* Public complaints
* Education and scholarships
* Jobs and skill training
* Emergency services

The platform is designed using Streamlit and Python.

## ▶ How to Run the Project

### Clone Repository

```bash
git clone https://github.com/meghanareddy0915-ops/ai-village-help-portal.git
```

### Open Project

```bash
cd ai-village-help-portal
```

### Install Requirements

```bash
pip install -r requirements.txt
```

### Run Application

```bash
streamlit run app.py
```
